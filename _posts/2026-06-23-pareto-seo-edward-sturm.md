## Os ensinamentos centrais do Pareto SEO

### 1. SEO técnico avançado não é para todo site
- Para a maioria dos sites, basta conectar o **Google Search Console**, enviar o sitemap e submeter URLs novas.  
- SEO técnico profundo só importa em casos específicos: sites enormes, SEO programático, migrações complexas.  

### 2. Google Search Console é a ferramenta central
- É gratuito e direto.  
- Submeta manualmente URLs novas ou editadas para acelerar indexação.  

### 3. Posições 3–20 são oportunidades imediatas
- Use o GSC para filtrar queries nessa faixa.  
- Crie páginas dedicadas ou expanda conteúdo existente para avançar posições.  

### 4. Os 5 pontos que resolvem 70% do SEO on-page
- Title tag  
- Meta description  
- URL  
- H1  
- Primeira frase do primeiro parágrafo  

### 5. Keywords de fundo de funil
- Menos concorrência, maior conversão.  
- Exemplo: “software de colaboração para equipes remotas com integração ao Slack”.  

### 6. Backlinks: menos do que você pensa
- Diretórios de nicho e plataformas de jornalistas já bastam para começar.

## ✅ O Filtro de Pareto: o que entra no seu 20%

| Faz isso (20%) | Não agora (80%) |
|----------------|-----------------|
| Configurar o GSC e enviar sitemap | Schema markup detalhado para sites simples |
| Keyword no título, meta, URL, H1 e primeira frase | Otimização de canonical tags |
| Analisar keywords em posições 3–20 no GSC | Keywords informacionais de alto volume |
| Criar páginas para keywords de fundo de funil | Link building em escala antes do básico |
| Submeter URLs novas manualmente no GSC | Análise de crawl budget para sites pequenos |
